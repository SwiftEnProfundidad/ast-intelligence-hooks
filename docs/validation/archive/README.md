# Validation Archive

This folder contains historical execution reports captured during framework rollout milestones.

These files are retained for audit traceability only and are not operational runbooks.

Active operational documentation lives in `docs/validation/README.md`.

Examples in this folder include:

- `skills-rollout-consumer-ci-artifacts.md`
- `skills-rollout-r_go-multi-platform-report.md`
- `private-actions-healthcheck.md`
