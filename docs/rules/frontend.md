# Frontend AST Rules

Esta sección documentará todas las reglas AST para **Frontend** (React / Next.js / TypeScript).

## Estructura

Cada regla seguirá este formato:

- **Id**: `frontend.<dominio>.<nombre>`
- **Severidad**: `critical | high | medium | low | info`
- **Descripción**
- **Ejemplo que viola la regla**
- **Ejemplo correcto**
- **Cómo desactivarla o ajustar severidad**
